よくある質問

Q: WebDAVサーバーとは何ですか？

WebDAVは、HTTPを拡張したファイル共有プロトコルです。NAS（Synology, QNAPなど）やクラウドストレージの多くがWebDAVに対応しています。院内のファイルサーバーとして利用できます。

Q: サーバーに接続できません

以下を確認してください：

  • URLが http:// または https:// で始まっているか
  • ユーザー名とパスワードが正しいか
  • 端末がサーバーと同じネットワークに接続されているか（ローカルサーバーの場合）
  • 設定画面の「接続テスト」ボタンで接続を確認

Q: 問診票のPDFはどこに保存されますか？

設定画面で指定したWebDAVサーバーのアップロードフォルダに保存されます。デフォルトは /monshin/ フォルダです。

Q: 送信に失敗した問診票を再送できますか？

はい。履歴画面で失敗した問診票の「再送」ボタンをタップしてください。

Q: 患者データのセキュリティは？

患者データは端末内のローカルデータベースに保存されます。パスワードはiOS Keychainに安全に保管されます。データは設定した保持期間後に自動削除されます。

Q: 画面の向きは変更できますか？

MonshinLiteは横向き（ランドスケープ）専用アプリです。問診入力の操作性を最適化するため、横向き固定で設計されています。

FAQ (English)

Q: What is a WebDAV server?

WebDAV is a file sharing protocol based on HTTP. Many NAS devices (Synology, QNAP, etc.) and cloud storage services support WebDAV. It can be used as an in-house file server.

Q: I can't connect to my server

Please check the following:

  • URL starts with http:// or https://
  • Username and password are correct
  • Your device is on the same network as the server (for local servers)
  • Use the "Connection Test" button in Settings to verify

Q: Where are the PDF questionnaires saved?

They are saved to the upload folder on the WebDAV server configured in Settings. The default folder is /monshin/.

Q: Can I resend a failed questionnaire?

Yes. Tap the "Retry" button on the failed record in the History screen.

Q: How is patient data secured?

Patient data is stored in a local database on the device. Passwords are stored securely in the iOS Keychain. Data is automatically deleted after the configured retention period.

Q: Can I change the screen orientation?

MonshinLite is a landscape-only app, designed to optimize the questionnaire input experience.
