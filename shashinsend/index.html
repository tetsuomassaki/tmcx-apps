ShashinSend

WebDAVで写真をワンタップ送信

Features

  • カメラで撮影してすぐにWebDAVサーバーへ自動送信
  • フォトライブラリから選択した写真も送信可能
  • 送信履歴の確認と失敗時の再送機能
  • HEIC → JPEG 自動変換・画質調整
  • 接続テスト機能で設定を簡単に確認
  • パスワードはKeychainに安全に保存

Use Cases

  • 業務写真をPCの共有フォルダに素早く転送
  • 検査結果や現場写真をリアルタイムで共有
  • NASやWebDAVサーバーに直接写真を保存
