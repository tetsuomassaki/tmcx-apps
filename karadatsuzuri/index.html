検査結果QRスキャンで、からだの変化を綴る

App Store

機能

  • SmartBlood対応医療機関の検査結果QRコードを1秒でスキャン
  • 手入力不要、正確に検査値を記録
  • カテゴリ別の検査結果一覧と基準値との比較バー表示
  • 検査値の時系列トレンドグラフで変化を可視化
  • 専門用語もやさしく解説する「わかる」タブ
  • 過去の検査記録を日付順に振り返れる履歴機能
  • Face ID / Touch ID によるアプリロック
  • 完全オフライン動作、データは端末内のみに保存
  • HL7 FHIR R4 / JLAC11 / UCUM 準拠、QRコードは ISO/IEC 18004 規格

こんな方におすすめ

  • 定期的に血液検査を受けている方
  • 自分の検査値の変化を長期的に把握したい方
  • プライバシーを重視し、クラウドにデータを預けたくない方
  • 紙の検査報告書の管理に悩んでいる方

免責事項

  • 本アプリは医療機器ではありません。診断・治療の目的には使用しないでください
  • 検査結果について気になることがあれば、必ず主治医にご相談ください

Features

  • Scan SmartBlood lab result QR codes in just 1 second
  • Accurate test value recording, no manual entry needed
  • Categorized test results with reference range comparison
  • Time-series trend graphs to visualize changes
  • Plain-language explanations of medical test items
  • Chronological history of all past test records
  • App lock via Face ID / Touch ID
  • Fully offline, all data stored on-device only
  • Compliant with HL7 FHIR R4, JLAC11, UCUM; QR symbol follows ISO/IEC 18004

Perfect For

  • Anyone who regularly undergoes blood tests
  • People who want to track their test values over time
  • Privacy-conscious users who prefer not to store health data in the cloud
  • Anyone struggling to manage paper lab reports
