MediPiro

QR スキャン or 手入力でラベルを Brother プリンタへ即印刷

App Store

機能

  • 患者氏名・ID を入力するとラベルプレビューがリアルタイム更新
  • 診察券・処方箋・調剤録などの QR コードをカメラで読み取り、患者情報を自動入力
  • Bonjour で同一 Wi-Fi 上の Brother プリンタを自動検出
  • 検出失敗時は IP アドレス手動指定（編集モード）に切替可
  • TCP 9100 で Brother プリンタへ直接送信、クラウド経由の遅延なし
  • DK-1201（29×90 mm ダイカット）/ DK-2205（62 mm 連続）に対応
  • 印刷成功後はフォームが自動クリアで連続発行に最適
  • 用紙切れ・カバーオープンなどプリンタ警告を画面表示
  • 印刷履歴を Face ID / Touch ID で保護
  • すべてのデータは端末内（Core Data）にのみ保存、外部サーバー送信なし

こんな現場で

  • クリニック・診療所での患者ファイル用ラベル
  • 調剤薬局での薬袋・在庫ラベル
  • 検査施設での検体ラベル
  • その他、Brother ラベルプリンタを利用した患者識別ラベル印刷業務全般

  • iPhone（iOS 18.0 以上）

  • 同一 Wi-Fi ネットワーク
  • 対応用紙: DK-1201 / DK-2205

Features

  • Live label preview that updates as you type the patient name and ID
  • Auto-fill patient info by scanning QR codes from medical cards or prescriptions
  • Bonjour-based auto discovery of Brother label printers on the same Wi-Fi
  • Manual IP entry mode as a fallback when discovery fails
  • Direct TCP/9100 transmission to the printer — no cloud round-trip
  • Supports DK-1201 (29 × 90 mm die-cut) and DK-2205 (62 mm continuous)
  • Form auto-clears after a successful print, optimized for batch printing
  • On-screen alerts for paper out, cover open, and other printer states
  • Print history gated by Face ID / Touch ID
  • All data stays on-device (Core Data); nothing is sent to an external server

Perfect For

  • Clinics and physician offices that print patient file labels
  • Pharmacies and dispensaries that print medication bag and inventory labels
  • Diagnostic labs that print specimen labels
  • Any setting that uses Brother label printers for patient identification

Requirements

  • iPhone running iOS 18.0 or later
  • A Brother label printer (e.g., QL-820NWB / QL-720NW) compatible with Brother Print SDK V4.13.0
  • Same Wi-Fi network as the iPhone
  • Paper: DK-1201 / DK-2205
